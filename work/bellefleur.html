bellefleurnailandbeauty.com

Belle Fleur Nail and Beauty needed a website for her at home nail and beauty business

The website had a lot of information to display and needed to be easy to navigate for mobile and larger devices. Using JavaScript for the mobile and desktop navigation I was able to make the one page site easy to navigate

I built the website for Belle Fleur Nail and Beauty using Bourbon's flexible lightweight Sass grid: Neat. I chose Neat grid as it is lightweight and needed a framework only for the grid. Using Neat's mixins also meant I could keep my markup semantic.
